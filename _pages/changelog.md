# Changelog

### `Latest`
# **Version 1.65.2**
- Get notifications about Org Agenda items
- Honor `org-hidden-keywords` set in local variables
- Add control to reset directory permissions
- Bug fixes

- Rendering improvements
- Bug fixes

- Bug fixes

<br>

## **Version 1.62.2**

- Bug fixes

<br>

## **Version 1.61.5**

- Additional proportional fonts now available
- Added Russian localization
- Bug fixes

- Open Org Mode files from URLs

- Bug fixes
